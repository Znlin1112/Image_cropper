# Image Cropper

This is a simple, browser-based image cropper generated with the help of Gemini. It features two modes to crop images into perfectly sized pieces without any white borders. All advice and feedback are welcome!

這是一個輕量級的網頁版圖片裁切工具，擁有兩種分割模式，能幫你精準算出完美比例不再需要手動切到死。

因為是純網頁應用程式，所有的圖片處理都在你的裝置本機端完成，完全不需要上傳到伺服器，安全又能保有隱私！

## ✨ 功能特色
* **兩種裁切模式：** 支援「指定等份數量」與「指定單塊像素」兩種方式。
* **白邊毀滅者：** 程式會自動進行完美置中微調，確保切出來的小圖尺寸一致且無白邊。
* **保留原圖格式：** PNG 透明背景不流失，JPEG 畫質不壓縮。
* **跨平台支援：** 只要有瀏覽器，電腦和手機都能直接使用。

使用網址：https://znlin1112.github.io/Image_cropper/

## 📱 如何在手機上當作 App 使用？

你可以將這個網頁直接「加入手機主畫面」，這樣它看起來、用起來就會像一個真正的獨立 App！

### 🍎 iOS (iPhone / iPad) 用戶
1. 使用 **Safari** 瀏覽器打開本專案的網址。
2. 點擊螢幕正下方的 **「分享」圖示**（一個方塊往上指的箭頭）。
3. 在跳出的選單中往下滑，找到並點選 **「加入主畫面」**。
4. 點擊右上角的「新增」，你的手機畫面上就會出現專屬的 App 圖示囉！

### 🤖 Android 用戶
1. 使用 **Chrome** 瀏覽器打開本專案的網址。
2. 點擊瀏覽器右上角的 **「⋮」(三個點選單)**。
3. 在選單中找到並點選 **「加到主畫面」**。
4. 點選「新增」，就能在手機主畫面上隨時點開來裁切圖片了！
